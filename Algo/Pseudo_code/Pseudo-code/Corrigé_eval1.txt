Exercice 1

VARIABLES
	longueur est un entier
	largeur est un entier
	perimetre est un entier
	aire est un entier
Debut du programme
	Ecrire "Saisir la longueur"
	Lire longueur
	Ecrire "Saisir la largeur"
	Lire largeur
	perimetre <-- (longueur + largeur)*2
	aire <-- longueur*largeur
	Ecrire "Perimetre du rectangle : ",perimetre
	Ecrire "l'aire du rectangle : ",aire
Fin du programme
